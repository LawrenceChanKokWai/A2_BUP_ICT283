\doxysubsection{Detailed Description}
\subsubsection*{template$<$class T$>$\newline
class Menu$<$ T $>$}

A class representing a menu interface for sensor data analysis. 

This class provides various options for processing and analyzing sensor data, including calculating averages, standard deviations, exporting data to files, and calculating sample Pearson correlation coefficients.
